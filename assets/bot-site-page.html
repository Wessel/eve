Notices

  1. Please note that, due to hardware restrictions, this bot is not able to run 24/7. If you would like to donate to support making the bot 24/7 in the future, please refer to my Ko-Fi (https://ko-fi.com/wessel)
  2. This model is far from perfect. This model has been trained on an uncurated dataset of text-image pairs and thus some prompts may not be accurate.
  3. By using EVE, you accept that prompt-image pairs may be saved with their respective authors (in the form of ID) for furthur training/catalogue purposes.
  4. Images may be presented as black squares, this is due to the NSFW filter. This filter is set to be strict to avoid any accidental slip-ups from happening.
